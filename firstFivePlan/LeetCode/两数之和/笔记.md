# 两数之和

给定一个数组**nums**和一个**整数目标值target**，请在该数组中找出合为目标值的两个整数，并返回他们的下标

```
输入：nums = [2,7,11,15],target = 9
输出：[0,1]
```
## 解决方法

1. 暴力求解

时间复杂度高 O(n^2)

2. hashmap

考虑hash表存储扫描后的数据结果

设置一个i扫描数组，每扫描一个数，判断概数与target的差值，去hashmap中查找是否存在这个差值，如果不存在，则将正在扫描的数字放入hashmap中去，直到找到这两个数相加等于target

hashmap中存贮的value值是该元素的下标，key为元素值

时间复杂度O(n)
