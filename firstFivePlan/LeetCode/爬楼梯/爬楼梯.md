# 假设你正在爬楼梯，需要n阶你才能到达楼顶，每次你可以爬一个或两个台阶，共有多少种不同的方法？ {ignore}

## 递归

fn = 
- fn = 1    n = 1; 
- fn = 2    n = 2;
- fn = f(n-1)+f(n-2)    n >= 3

上述时间复杂度太高，进行优化

## hashmap

f(6) = f(5) + f(4)
f(5) = f(4) + f(3)
f(4) = f(3) + f(2)

f(3),f(4)等求解多次

制造hashmap，每求解一次后放入hashmap中，下次使用查表搜索  O(n)

## 循环

自底向上

f(3) = f(2) + f(1)
f(4) = f(3) + f(2)  O(n)
